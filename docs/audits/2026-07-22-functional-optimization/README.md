## 用户主流程

```mermaid
flowchart LR
  A[拖入素材] --> B{已有底图?}
  B -- 否 --> C[选择底图]
  C --> D[自动续接素材放置]
  B -- 是 --> D
  D --> E[编辑与多选]
  E --> F[锁定层受保护]
  F --> G[自动保存]
  G --> H{主文件可用?}
  H -- 否 --> I[校验备份并恢复]
  I --> J[保留已知良好备份]
  H -- 是 --> J
```
